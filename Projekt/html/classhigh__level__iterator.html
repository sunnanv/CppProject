Detailed Description

template<typename T1, typename T2>
class high_level_iterator< T1, T2 >

An random access iterator class. The high_level_iterator allows a user to iterate over two iterators as if they where one. Does not copy the containers, it uses pointers.
T1 and T2 is the types of the containers (e.g vector<int>). They do not have to be of the same type.

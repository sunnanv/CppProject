Detailed Description

template<typename T1, typename T2>
class concatenation< T1, T2 >

A class for concatenation two containers without copying them. After concatenation it is possible to iterate over the containers as if they where one. Supports const containers.
T1 and T2 is the types of the two containers (they do not need to be of the same type).
